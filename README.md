Global Financial Trading Dashboard
About the Project

The Global Financial Trading Dashboard is an interactive web application built using Python, Streamlit, Pandas, and Plotly.

This project allows users to:
Select Country
Select Company
Filter by Date
View Stock Trends (Line Chart & Candlestick Chart)
See Stock Details (Open, High, Low, Close, Volume)

 The dashboard is designed like a trading platform to make stock data easy to understand and visually attractive.

Technologies Used :
1.Python
2.Pandas
3.Plotly

Features :
Interactive Sidebar Filters
Multiple Colored Line Chart
Candlestick Chart
Stock Metrics Panel
Data Table View
Download Option

Purpose
This project is created for academic purposes to demonstrate financial data analysis and dashboard development using Python.
